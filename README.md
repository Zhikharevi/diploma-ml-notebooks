# Diploma ML Notebooks

Репозиторий содержит ноутбуки и вспомогательные скрипты для дипломного проекта по прогнозированию времени выполнения задач. Основная цель проекта - сравнить классические ML-модели, DL-модели, ансамбли и признаки, полученные с помощью LLM, на задаче регрессии `duration_hours`.

## Что внутри

- `ML_*` - baseline, tuning и ансамбли классических моделей.
- `DL_*` - baseline, tuning и ансамбли нейросетевых моделей.
- `ML_meta_*`, `Dl_cluster*` - эксперименты с мета-признаками и кластеризацией.
- `Llm_*`, `LLM_*`, `09_llm_*` - эксперименты с LLM-признаками, OFAT-подбором prompt-параметров, воспроизводимостью и provider-level ансамблями.
- `Post_OFAT_*` - финальные эксперименты поверх OFAT-артефактов.

## Данные и артефакты

Исходные данные, обученные веса, LLM-кэши и большие экспериментальные артефакты не публикуются в репозитории. Они содержат внутренние рабочие данные и производные результаты, которые нельзя раскрывать публично из-за требований информационной безопасности и конфиденциальности компании Яндекс.

Поэтому в ноутбуках пути к данным и артефактам вынесены в переменные окружения. Основная переменная для датасета:

```bash
export DATA_FINALL_PATH=/path/to/data_finall.csv
```

Для отдельных экспериментов могут использоваться дополнительные переменные окружения с путями к локальным артефактам.

## Воспроизводимость

Ноутбуки очищены от outputs, локальных абсолютных путей и служебных Colab/Drive-настроек. Для полного воспроизведения результатов нужен доступ к закрытым данным и локальным артефактам экспериментов.

Публичная часть репозитория предназначена для демонстрации методологии, структуры экспериментов и кода моделирования без раскрытия чувствительных данных.
